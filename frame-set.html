Html元素重置

在 reset.css 里，XHTML, HTML4和HTML5的元素都进行了样式重置，这样你可以更加自由的定制个性的CSS样式了。

列表项

用 ul 创建一个无序列表，或者用 ol 创建一个有序的列表。重置文件 reset.css 中去除了列表自带的点和内间距等样式，需要的话可自行定义。

例子

  • List Item 1
  • List Item 2
    • Sub List Item 1
    • Sub List Item 2
    • Sub List Item 3
  • List Item 3
  1. List Item
  2. List Item
    • Sub List Item
    • Sub List Item
    • Sub List Item
  3. List Item
<ul>
  <li></li>
  <li>
    <ul>
      <li>…</li>
      <li>…</li>
    </ul>
  </li>
  <li></li>
