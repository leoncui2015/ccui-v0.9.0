让我们开始把！

这是一个教我们应用CCUI框架来快速搭建网站界面的入门手册

你首先需要下载框架

所有代码文件都能在 云盘里 找到

CCUI源文件

文件包含所有的HTML,CSS,JS图片和字体文件。你也可以改进CCUI。在此之前，你最好了解一些基础的前端技术。

下载CCUI源文件

文件结构

下载完成后，解压即可看到所有的文件。CCUI保持在最轻型的状态，让你在开发的时候有更大的发展空间，同时我们也封装了一些组件的样式。这些都可以在 CSS 文件里找到并修改。

文件 描述
/css 包含主要的框架和样式文件
/js 包含各种Javascript插件和Jquery框架文件
/fonts 包含Awesome 图标文件
/images 包含CCUI展示网站应用到的图片文件

设置你的源代码文件

你会在根目录找到你需要设置和修改的所有文件 

<!DOCTYPE html>
<html>
	<head>
		<link rel="stylesheet" href="css/yourfile.css" />
		<script src="js/yourfile.js"></script>
		<title></title>
	</head>
	<body>
	</body>
</html>

Viewport标签

CCUI采用了自适应网格系统，所以在 head 标签内包含了viewport标签。 

<meta name="viewport" content="width=device-width, initial-scale=1.0">

支持的浏览器

CCUI不仅支持最新的浏览器还兼容拥有较大用户量的IE较低版本的浏览器。我们支持最新的桌面和手持设备显示器，老式的IE浏览器只做到兼容排版和布局，动态及美观的效果不能完全支持。我们支持如下最新版本的5种浏览器。

Chrome Firefox Internet Explorer Opera Safari
Mac OS X
Windows
Android
iOS
