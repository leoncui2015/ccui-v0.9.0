提示框

提示框给用户提供了各种动作的反馈。这些反馈能帮助用户解释他们在一个事件上是成功还是失败了。

提示构建块

所有的提示框建立在 .alert 类, 并且它也有很多的修饰类。 举个例子，下面是一个右侧带图标的带 a-is-primary 主要信息类的提示。

例子

这是一些 重要信息 请注意。 

<div class="alert a-is-info">
 <p>这是一些 <strong>重要信息</strong> 请注意。<span class="fa fa-info"></span></p>
</div>

使用Jquery来移除或隐藏提示框

使用DOM，可以灵活地达到移除提示框或验证的目的，例如，只隐藏他们，所以他们不可见。下面是使用jQuery的一个例子。 

$('.close').click(function(e) {
  var elem = $(e.currentTarget).parents('.alert-removed');
  elem.removeClass('in');
  
  setTimeout(function () {
    elem.remove();
  }, 450);
});

// This just hides the element
$('.close').click(function(e){
  var hiding = $(e.currentTarget).parents('.alert-hide');
  hiding.removeClass('in');
});

例子

点击 X 来 隐藏这个提示框. 使用类 .alert-hide 来因隐藏这个提示框 ×
