表单

内容输入表单 (input, textarea, select, 等) 需要 .form-element 类。 这个类给这些元素他们的样式。

禁用元素使用类 is-disabled

基本的表单例子

输入元素需要你设置宽度。

例子

文件上传格式大小说明

<form>
 <div class="form-group">
  <label for="emailID">邮箱地址</label>
  <input type="email" class="form-element" id="emailID" placeholder="你的邮箱地址">
 </div>
 <div class="form-group">
  <label for="passwordID">密码</label>
  <input type="email" class="form-element" id="passwordID" placeholder="你的密码">
 </div>
 <div class="form-group">
  <label for="someFile">文件上传</label>
  <input type="email" class="form-element" id="someFile">
  <p>文件上传格式大小说明</p>
 </div>
 <div class="form-group">
  <label>
    <input type="checkbox"> 选项表单
  </label>
 </div>
 <button type="submit" class="btn btn-primary">提交</button>
</form>

基本的表单元素

这是一个基本的输入表单。确保在 input 标签上加上 form-element 类。

例子

<input type="text" class="form-element">

大的表单元素

例子

在 input 标签上加上 form-element-lg 类。

<input type="text" class="form-element-lg form-element">

小的表单元素

例子

在 input 标签上加上 form-element-sm 类。

<input type="text" class="form-element-sm form-element">

无法输入的表单元素

例子

在任何表单元素标签上加上 is-disabled 类使表单无法输入。

<input type="text" class="form-element-sm form-element is-disabled">
