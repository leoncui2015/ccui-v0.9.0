媒体查询

媒体查询是CSS3模块中允许你呈现的内容在不同的屏幕分辨率下自适应。 媒体查询是实现网页响应式设计的三个过程中的一环。这些功能对不同的宽度和高度的内容会非常有用。它可以针对特定分辨率范围的显示设备，而不必改变内容本身。

提示

媒体查询的基本格式 

@media screen and (min-width: 979px) { ... }

提示

CCUI网格系统在960网格系统基础进行了扩展。主要的媒体查询分成桌面，平板和手机设备。

命名 范围 显示器
desktop-lg Screen and (min-width: 1200px) 宽屏显示器
desktop Screen and (min-width: 980px) and (max-width: 1199px) 桌面显示器
tablet Screen and (max-width: 979px) 平板及PAD显示器
mobile Screen and (max-width: 736px) 手机显示器

元素隐藏

有时候我们需要在不同的显示器上隐藏一些内容，这就需要用到元素隐藏。它使用一些特定名称的类在特定的显示器上隐藏这些内容。

类 描述
desktop-lg-hidden 使用这个类在宽屏的显示器隐藏内容
desktop-hidden 使用这个类在桌面的显示器隐藏内容
tablet-hidden 使用这个类在平板的显示器隐藏内容
mobile-hidden 使用这个类在手机的显示器隐藏内容 
<div class="mobile-hidden">
  { ... }
</div>

元素显示

同样，我们使用一些特定名称的类在特定的显示器上显示一些内容。

类 描述
desktop-lg-show 使用这个类在宽屏的显示器显示内容
desktop-show 使用这个类在桌面的显示器显示内容
tablet-show 使用这个类在平板的显示器显示内容
mobile-show 使用这个类在手机的显示器显示内容 
<div class="mobile-show">
  { ... }
</div>
