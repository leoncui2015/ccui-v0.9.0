表单输入组

表单输入组是表单控件的一个扩展，比如，你需要在一个文本输入域 input 的前面或后面放置一个按钮。

输入组由 .form-element-group 类来创建

基本的样式

基于文本的附加元素，使用类 .form-element-extra 。在这个例子中它在末端。

使用 .form-border-left 来去除左侧元素的右侧圆角， .form-border-right 正好相反，它应用在右侧的元素。

例子

额外的表单元素 
<div class="form-element-group">
 <input type="text" class="form-element form-border-left" placeholder="占位符">
 <span class="form-element-extra form-border-right">额外的表单元素</span>
</div>

现在的在开始的位置。

例子

额外的表单元素 
<div class="form-element-group">
 <span class="form-element-extra form-border-left">额外的表单元素</span>
 <input type="text" class="form-element form-border-right" placeholder="占位符">
</div>

尺寸

大尺寸和小尺寸的表单使用类 .form-element-group-lg 和 .form-element-group-sm

例子

大的表单集合

基本的表单集合

小的表单集合 
<div class="form-element-group-lg">
 <input type="text" class="form-element form-border-left" placeholder="占位符">
 <span class="form-element-extra form-border-right">大的表单集合</span>
</div>

<div class="form-element-group">
 <input type="text" class="form-element form-border-left" placeholder="占位符">
 <span class="form-element-extra form-border-right">基本的表单集合</span>
</div>

<div class="form-element-group-sm">
 <input type="text" class="form-element form-border-left" placeholder="占位符">
 <span class="form-element-extra form-border-right">小的表单集合</span>
</div>

添加元素

你可以添加类似多选，单选的表单，图标等元素。

例子

<div class="row-fluid">
 <div class="grid6">
  <div class="form-element-group">
   <input type="text" class="form-element form-border-left" placeholder="占位符">
   <span class="form-element-extra form-border-right"><input type="checkbox"></span>
  </div>
 </div>
 <div class="grid6">
  <div class="form-element-group">
   <input type="text" class="form-element form-border-left" placeholder="占位符">
   <span class="form-element-extra form-border-right"><input type="radio"></span>
  </div>
 </div>
</div>

<div class="row-fluid">
 <div class="grid6">
  <div class="form-element-group">
   <input type="text" class="form-element form-border-left" placeholder="占位符">
   <span class="form-element-extra form-border-right"><span class="fa fa-check"></span></span>
  </div>
 </div>
 <div class="grid6">
  <div class="form-element-group">
   <input type="text" class="form-element form-border-left" placeholder="占位符">
   <span class="form-element-extra form-border-right"><span class="fa fa-arrow-right"></span></span>
  </div>
 </div>
</div>

添加按钮

你可以在表单元素集合的开始或末端位置添加按钮。

确保你的按钮在类 .form-group-btn 里

例子

<div class="form-element-group">
     <input class="form-element form-border-left" type="text" />
     <span class="form-group-btn">
      <button class="btn btn-default form-border-right">基本的按钮</button>
     </span>
</div>
