表格

表格兼容性较好，使用也方便。我们只需要在表格上添加 .table 类即可。

基本的表格

CCUI的表格样式简洁，下面列举一些常用的样式。

例子

菜单 口味 价格
玉米煎饼 一级棒 ￥6.00
口水虾 非常棒 ￥60.00
披萨 最棒 ￥40.00 
<table class="table">
    <tr>
      <th>菜单</th>
      <th>口味</th>
      <th>价格</th>
    </tr>
    <tr>
      <td>玉米煎饼</th>
      <td>一级棒</th>
      <td>￥6.00</th>
    </tr>
    <tr>
      <td>口水虾</th>
      <td>非常棒</th>
      <td>￥60.00</th>
    </tr>
    <tr>
      <td>披萨</th>
      <td>最棒</th>
      <td>￥40.00</th>
    </tr>
</table>

表格修饰类

下面是一些你可以用到表格行上的类。

类 描述
is-success 添加成功的颜色到表格行上
is-primary 添加主要的颜色到表格行上
is-info 添加信息的颜色到表格行上
is-warning 添加警告的颜色到表格行上
is-danger 添加危险的颜色到表格行上
