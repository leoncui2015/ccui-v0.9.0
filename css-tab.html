标签

创建标签使用类 btn-label 。

基本的标签

基本的标签样式。按钮样式的标签不需要 类 btn。

按钮标签用类 btn-label 创建，并且有一些修饰类，大致的如下面所示。

基本的标签 
<span class="btn-label">基本的标签</span>

标签按钮修饰类

添加一些修饰类来改变一些特殊响应的UI。

基本的标签 主要的标签 成功的标签 信息的标签 警告的标签 危险的标签 
<span class="btn-label">基本的标签</span>
<span class="btn-label primary">主要的标签</span>
<span class="btn-label success">成功的标签</span>
<span class="btn-label info">信息的标签</span>
<span class="btn-label warning">警告的标签</span>
<span class="btn-label danger">危险的标签</span>
